# LDD Iframe Upscaler

A tiny React + Vite + Tailwind app that shows a branded splash screen while loading an external site in an iframe.

- Splash shows your logo and the text: “Loading LavenderDragonDesign's Upscaler — Web React App”
- Fades out on iframe `onLoad` or after a 4s safety timeout
- Fullscreen iframe without scrollbars or borders

## Quick start

```bash
npm i
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
